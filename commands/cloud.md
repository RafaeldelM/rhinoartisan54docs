# Cloud

<figure><img src="../.gitbook/assets/image 12.png" alt=""><figcaption></figcaption></figure>

### <img src="../.gitbook/assets/FILE_USER_FOLDER_APP.png" alt="" data-size="line">Open from Cloud&#x20;

This command opens a new window that shows all the designs and labels you keep in 2Shapes Design.

The 2Shapes Designs window allows you to click on the various labels you have created on 2Shapes Design to navigate through them. You can also click on any design you see in this window, and it will be imported to your open 2Shapes for Rhino file.

![](<../.gitbook/assets/Screenshot 2022-06-28 120738.jpg>)

### <img src="../.gitbook/assets/FILE_USER_FOLDER_APP.png" alt="" data-size="line">Save on Cloud&#x20;

This command saves your file as a design on 2Shapes Design, along with the rest of your designs.

Once you run this command, a window will open where you can Name the resulting design, and in which label you want to store it. Click on the Save button to start the uploading process, or the Cancel button to stop this command.

![](<../.gitbook/assets/image (121).png>)

### <img src="../.gitbook/assets/FILE_GALLERIES.png" alt="" data-size="line">Galleries

Running this command will open a new window in your internet browser that shows the galleries and labels you keep in 2Shapes Design.

![](<../.gitbook/assets/image (236).png>)

### <img src="../.gitbook/assets/GEM_NONCERTIFICATE.png" alt="" data-size="line">Small Diamonds

{% hint style="warning" %}
This feature requires having at least one active diamond group on your 2Shapes organization.
{% endhint %}

With this command, you can add existing small diamonds from your 2Shapes Pricing diamond groups to your 2Shapes for Rhino file.

![](<../.gitbook/assets/image (186).png>)

Running this command opens a window listing all your diamond groups with small diamonds in it, you can click on them to see all their entries.

With the Diamonds and Gemstones window open, you can see all the available Small Diamonds, and also access Diamonds and Gemstones on their corresponding tabs. Clicking the **+** button on the left of each entry allows you to add them to your design.

After adding the small diamond to your file, it will be listed on the Outliner toolbar.

### <img src="../.gitbook/assets/SMART_ELEMENT (2).png" alt="" data-size="line">Elements <a href="#elements" id="elements"></a>

This command opens the Elements window, where all your elements saved on your 2Shapes account are kept.

Once you run this command, you can click on the labels on the left to open them or click on an element to add them to your design on 2Shapes for Rhino.

The chosen element will be listed on the Outliner toolbar.



### <img src="../.gitbook/assets/unnamed.png" alt="" data-size="line">Bespoke create <a href="#create" id="create"></a>

With this command, you can select any geometry and make 2Shapes recognize it as an Element.

Once you run this command, you will be asked to select which object you wish to be a Bespoke Element. After clicking on it, you will need to type in a name for it.

After that, your Bespoke Element will be listed on the Outliner toolbar with the name you have set.

### <img src="../.gitbook/assets/BESPOKE_SYNC.png" alt="" data-size="line">Bespoke sync <a href="#sync" id="sync"></a>

With this command, you can synchronize the changes made to a Bespoke Element, and save it on 2Shapes Design, along with your other elements.

After running this command, you will be asked to select the Bespoke Element you want to synchronize. After doing so, it will start the process and finish the command.
